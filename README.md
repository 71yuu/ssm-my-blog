## Yuu Blog

这是一个使用 Maven 构建的多模块化的 SSM（Spring + Spring MVC + MyBatis）项目。



## 开发环境

- Intellij IDEA 2018.3.24
- JDK 8
- Tomcat 8
- MySql 5.7
